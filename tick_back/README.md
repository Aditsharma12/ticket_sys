# KIET Ticket Management System 🎟️🎓  

## Overview  
This project is designed to help manage tickets for various events such as seminars, lectures, and workshops at KIET. It aims to streamline the process of ticket booking, registration, and management.

## Architecture Overview  
```
+------------------+
|   User Interface  |
+------------------+
         |
         V
+------------------+
| Application Logic |
+------------------+
         |
         V
+------------------+
|    Database      |
+------------------+
```

## Features  
- **User Registration:** 📋  
  Allows users to create an account and manage their tickets.  
- **Event Management:** 📅  
  Admin can create and manage events, including ticket pricing.  
- **Ticket Booking:** 🎫  
  Users can easily book tickets for their desired events.  
- **Real-time Updates:** 🔄  
  Users receive notifications about event changes.  

## Installation Instructions  
1. Clone the repository:  
   ```bash  
   git clone https://github.com/Aditsharma12/KIET_Ticket.git  
   ```  
2. Install dependencies:  
   ```bash  
   npm install  
   ```  
3. Start the server:  
   ```bash  
   npm start  
   ```
